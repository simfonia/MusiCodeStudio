# 2026-05-02 通訊架構重大突破：成功切換至 HTTP POST 模式

### 完成事項
1.  **環境配置定案**：確認 VS Code 全域使用 `Visual Studio 17 2022` 產生器，並鎖定 VS 2022 amd64 Kit 以解決編譯器偵測問題。

    *   放棄因模組缺失（juce_cryptography）導致握手失敗的 WebSocket。

    *   C++ 端 `CommandServer` 執行緒正確解析指令並觸發 UI 狀態變更（PLAYING/STOPPED）。

*   **端口**：9001 (避免與常用 8080 衝突)。
*   **通訊協定**：HTTP/1.1 POST + JSON Body。
*   **引擎狀態**：成功脫離 `te::Edit` 初始化依賴，優先確保「通訊神經系統」穩定。
