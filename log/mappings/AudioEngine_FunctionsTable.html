AudioEngine 功能與檔案對照表

功能模組 實作檔案 關鍵函數 / 指令 外部依賴 備註
音訊引擎核心 Source/AudioEngine.cpp AudioEngine(), play(), stop() te::Engine, te::Edit 管理播放狀態與 BPM。
HTTP 伺服器 Source/HttpServer.cpp HttpServer::run(), handleRequest() juce::StreamingSocket 接收指令並解析。
指令解析 (JSON) Source/HttpServer.cpp processCommand() juce::JSON 解析 JSON 指令，對應前端 action 欄位。支援 transport_play, transport_stop, set_bpm。
UI 狀態顯示 Source/HttpServer.cpp updateStatus() juce::Label 同步引擎狀態至視窗文字。
主視窗管理 Source/MainWindow.cpp MainWindow() juce::DocumentWindow 初始化 AudioEngine 並傳入 HttpServer。
應用程式框架 Main.cpp initialise(), shutdown() juce::JUCEApplication 管理 MainWindow 生命週期。
