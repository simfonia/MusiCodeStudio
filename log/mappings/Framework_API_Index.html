本專案關鍵點：
  • 初始化順序： 必須先呼叫 ensureContextAllocated()，否則無法獲取有效的 InputDeviceInstance。
  • 濾波器陷阱： 直接將 0-1 寫入 4OSC 的 filterFreq 會被視為 Hz，導致靜音。必須進行範圍轉換。
  • 設備開啟： 必須在 JUCE 層級明確 setMidiInputDeviceEnabled(true) 並註冊回呼，訊號才不會漏失。
