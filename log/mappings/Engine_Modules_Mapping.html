MusiCodeStudio 後端引擎模組對照表 (C++)

功能模組 關鍵檔案 核心職責 關鍵函數 / API 備註
Audio Engine Core AudioEngine.cpp/h 音訊引擎生命週期、Edit 管理、播放控制。 play(), stop(), setupTestScene() 封裝 te::Engine 與 te::Edit。
IPC Server HttpServer.cpp/h 監聽 Port 9001，處理來自前端的 JSON 指令。 registerHandlers(), processCommand() 使用 juce::StreamingSocket。
IPC Routing CommandRouter.cpp/h 解析 JSON 指令並分發至 AudioEngine。 processCommand() 實現代碼重用，支援 HTTP 與 Native IPC。
Plugin Manager PluginController.cpp/h 管理插件視窗生命週期、參數即時同步。 showPluginWindow(), setPluginParameter() 處理 4OSC 全域搜尋與 ValueTree 同步。
Native / Web UI MainWindow.cpp/h, MainComponent 承載原生視窗與 WebView2，處理 UI 與音訊初始化。 setContentOwned(), postToBackend 整合 Native IPC (Hybrid 模式)。
