MusiCodeStudio 通訊協議 (HTTP/JSON)

Action 指令 參數結構 功能描述 觸發點 (UI)
transport_play 無 啟動播放，若已在結尾則自動重頭播放。 Play 按鈕
transport_stop 無 停止播放，重設指標至 0.0。 Stop 按鈕
set_bpm { value: number } 調整全局 BPM。 BPM 輸入框
show_plugin_window { track: number } 開啟指定音軌的第一個插件原生視窗。 音軌插件插槽
set_plugin_param { pluginName: string, paramID: string, value: number } 實時控制插件參數 (0.0 ~ 1.0)。 插件架滑桿
