TrackManager 模組

位置：engine/MusiCodeEngine/Source/TrackManager.h

功能：提供音軌 ID (EditItemID) 的定位、轉換與元數據封裝。解決音軌順序變動導致的索引位移問題。

主要方法

stringToID(const String& idString)

將前端傳來的 UUID 字串轉換為 te::EditItemID。

findAudioTrackByID(te::Edit& edit, te::EditItemID id)

在指定 Edit 中尋找對應 ID 的音軌指針。

getTracksInfo(te::Edit& edit)

返回一個 juce::var (Array of Objects)，包含所有音軌的 ID、名稱與當前索引，供前端渲染軌道列表。

範例代碼

// 前端傳來指令 { "action": "...", "trackID": "1001" }
auto id = MusiCode::TrackManager::stringToID(params["trackID"]);
auto* track = MusiCode::TrackManager::findAudioTrackByID(edit, id);
