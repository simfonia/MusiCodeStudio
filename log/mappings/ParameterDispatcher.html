ParameterDispatcher 模組

位置：engine/MusiCodeEngine/Source/ParameterDispatcher.h

功能：負責插件參數的標準化對接、執行緒同步與持久化。核心實作了「雙重更新同步模式」。

主要方法

setParameter(te::Plugin::Ptr plugin, const String& id, float normalizedValue)

設定單一插件參數。會自動切換至 Message Thread 執行。

  • 對 AutomatableParameter 進行 0-1 到 Raw 的範圍轉換。
  • 同步更新 ValueTree 屬性以確保 UI 與持久化一致。
  • 支援 Fallback 模式，處理離散屬性（如 4OSC 的 filterType）。

setParameters(te::Plugin::Ptr plugin, const std::map& params)

批量設定參數。常用於測試場景或預設值加載。

範例代碼

// 將 4OSC 的濾波頻率設為中點 (0.5)
MusiCode::ParameterDispatcher::setParameter(my4Osc, "filterFreq", 0.5f);
